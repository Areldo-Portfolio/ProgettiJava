GiocoDellImpiccato
Descrizione
Applicazione Java console che riproduce il gioco dell'impiccato. L'utente può indovinare la parola completa o una lettera per volta, con feedback in tempo reale.

Caratteristiche
Scelta tra indovinare parola o lettera.

Gestione robusta degli input con validazioni.

Visualizzazione dinamica della parola con lettere scoperte e trattini.

Controllo automatico della vittoria.

Tecnologie e concetti
Java OOP, uso di classi, metodi e override (toString()).

Collezione HashSet per tracciare lettere trovate.

Gestione input da console con Scanner e try-catch.

Cosa dimostra il progetto
Capacità di progettare e sviluppare un’applicazione interattiva da console.

Attenzione alla gestione degli errori e UX testuale.

Uso efficace di strutture dati e logiche di controllo.

Come eseguirlo
Compilare e lanciare la classe Main. Seguire le istruzioni a schermo.
