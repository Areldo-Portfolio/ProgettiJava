Progetto: Rubrica Contatti in Java
Descrizione:
Ho sviluppato una semplice applicazione console in Java per gestire una rubrica telefonica. Il progetto consente di aggiungere, visualizzare e rimuovere contatti, implementando l’interazione con l’utente tramite input da tastiera.

Funzionalità principali:

Aggiunta di nuovi contatti con nome e numero telefonico

Visualizzazione della lista completa dei contatti salvati

Rimozione di contatti selezionati tramite indice

Gestione del flusso tramite menu testuale intuitivo

Tecnologie e concetti utilizzati:

Programmazione orientata agli oggetti (classi, metodi, incapsulamento)

Utilizzo di collezioni Java (ArrayList) per memorizzare dati dinamici

Gestione input da tastiera con Scanner

Controllo del flusso con cicli while e strutture switch

Override del metodo toString() per una stampa leggibile degli oggetti

Obiettivi del progetto:

Approfondire la programmazione Java di base e la gestione di strutture dati

Comprendere il ciclo di vita di un’applicazione interattiva da console

Sviluppare abilità nell’organizzazione e modularizzazione del codice
